# 🧭 Compass OS

A completely serverless, AI-powered personal Operating System built entirely within a single, highly-optimized `.html` file. 

Compass OS bypasses traditional heavy backends by utilizing a **"Bring Your Own Key" (BYOK)** architecture, allowing for infinite scalability with zero hosting costs. All data is cryptographically secured via local browser storage, offering an enterprise-grade, privacy-first user experience.

### ⚡ Core Features
* **Zero-Latency UI:** Built with raw HTML/CSS/JS and Tailwind utilities for instant loading.
* **Multi-Node AI Gateway:** Seamlessly switch between Google Gemini, OpenAI, or decentralized free nodes.
* **True Multimodal Vision:** Drag-and-drop image analysis using the holographic visual core.
* **Web Grounding:** Real-time internet access for the AI to fetch live data and cite web sources.
* **Local Data Matrix:** 100% private data storage. Features an Activity Heatmap, Focus Timers, Task Strategy, and a Quantum Finance Simulator.
* **Hardware Biometrics:** Secure your session using TouchID or Windows Hello via WebAuthn protocols.

### 🚀 Getting Started
Compass OS requires zero installation or server setup. 
1. Simply visit the live deployment link.
2. Head to the **Settings** panel to configure your preferences, set up biometrics, or connect your premium API keys.

### 🛡️ Privacy & Security
This application is strictly front-end. It does not communicate with any external databases. Your API keys, notes, and metrics are stored exclusively in your browser's local enclave (`localStorage`).
